# Customizing Your Bot

Make your bot feel like a natural part of your brand. You can change its name, personality, colors, and welcome messages to ensure it provides the exact user experience you want.

![Bot Settings page showing all customization options](/docs-images/bot-settings.png)

## General Settings

Navigate to **Bot Settings** in the sidebar. This page is divided into several sections.

### Bot Name
This is the name visitors will see at the top of the chat window.
*Example: "Acme Corp Support" or "Help AI"*

### AI Personality (System Prompt)
This is arguably the most important setting. The personality prompt tells the AI how it should behave and respond.

![Personality Prompt text area showing a custom instruction](/docs-images/bot-settings.png)

> [!TIP]
> Be specific in your prompt.
> *Bad Prompt:* "You are a helpful assistant."
> *Good Prompt:* "You are a customer support agent for Acme Shoes. Always be polite, use emojis occasionally, and if you don't know the answer, tell the user to email support@acmeshoes.com."

### Welcome Message
This is the first message the bot sends when a user opens the chat widget. Use this to set expectations or direct users on what to ask.

## Appearance Settings

You can fully customize the look and feel of the chat widget to match your website's design.

![Appearance Settings section showing color pickers and uploading a logo](/docs-images/bot-settings.png)

### Brand Color
Click the color picker or enter a hex code to set the primary color for the chat interface. This color is used for the header background, user message bubbles, and action buttons.

### Bot Avatar/Logo
Upload a small square image to represent your bot. This will appear next to the bot's messages in the chat.
*Recommended size: 256x256px.*

### Chat Icon
By default, the widget uses a standard chat bubble icon. You can upload a custom icon to replace the default bubble that triggers the chat window on your website.

> [!NOTE]
> Any changes you make here are saved instantly and will automatically reflect on your live website widget without needing to update the embed code.
